git - githyub dersleri

- git init ile klasörümüzde giti başlatırız
- git add . komutu ile yaptığımız değişikleri working directoryden  stagin aarea ya taşırız
- git status ile commit öncesinde nerded oldugumuzu görürüz
- git commit -m "v1" komutu ile değişiklikleri staging areadan commit store a taşırız ve bir versiyon oluştururuz
- git commit --amend -m "v3" ile mesajı değiştirebiliriz
- git diff ile working space'deki değişikleri görebiliriz
- git show hashcode ile de commit store'deki değişiklikleri görebilirz
- git remote add origin url diyerek bir remote repostory adres tanımlamış oluruz
- git push -u origin master ilede dosyalarımızı push ederiz 
- git pull diyrek remote repostory'deki son halini alırız

git branch ile branchleri listeleriz
git branch --all ile remotedekibranchleride listeleriz
git branch name diyerek yeni bir branch oluşturunuz
git merge anatherbranch yazarsak branchleri birleştirmiş oluruz
git branch -d name yazarak localdeki bir brancı sileriz 

git stash ile değişikler gecici olarak hafızaya alınıır
